FAQ

Common questions about the TaxBro invoice maker.

This first version is intentionally practical: fast, light, and export-focused.

Does this save invoices online?

Not yet. This version saves drafts in your current browser through local storage, which keeps deployment simple and works on GitHub Pages.

How do I get a PDF?

Use the Print / PDF button. Your browser's print dialog lets you save the invoice preview as a PDF with the clean sheet layout.

Does it support IGST versus CGST and SGST?

Yes. The builder compares the seller GSTIN state code with place of supply or buyer GSTIN state code and switches the tax split automatically.

Can I replace the logo and signature?

Yes. Upload both from the form and the preview will update immediately. If you do nothing, the TaxBro logo remains as the default placeholder.
